## Description

This is an updated portfolio page to include additional materials and projects, building toward being employer-competitive. The portfolio showcases full-stack web development projects and provides professional contact information and materials for potential employers.

---

## Usage

Provide an updated portfolio to help potential employers evaluate my work and suitability for future opportunities. The portfolio serves as a central hub for showcasing technical skills, completed projects, and professional credentials.

## Portfolio Requirements

The updated portfolio includes:

- [x] Display full name, email, and phone number
- [x] Links to GitHub profile and LinkedIn profile
- [x] Link to a downloadable PDF resume
- [x] Display at least 2 examples of student work from deployed projects or homework assignments
  - [x] Project title
  - [x] Link to the deployed version
  - [x] Link to the GitHub repository
  - [x] Screenshot of the deployed application
- [x] Polished, mobile-responsive user interface

## Technology

### Portfolio Site:
* **HTML** (95.0%) - Structure and content
* **CSS** (5.0%) - Custom styling
* **Bootstrap** - CSS framework for responsive design
* **jQuery** - Required by Bootstrap
* **JavaScript** - Interactive functionality

### Featured Projects Utilize:
* **Foundation** - Alternative CSS framework
* **Node.js** - Server-side runtime
* **npm** - Package management
* **Jest** - Testing framework
* **MySQL** - Database management
* **Express.js** - Web application framework
* **Express Handlebars** - Templating engine
